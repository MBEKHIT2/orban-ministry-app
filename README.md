SMSV Orban Ministry App – Description
The SMSV Orban Ministry app is a mobile-friendly, offline-capable tool designed to streamline the preparation of Orban dough batches for ministry. Built from real production data, the app enables users to select a day and instantly view all required quantities and steps for each batch.

Key Features:

Day Selector: Choose the day to automatically populate batch requirements based on historical data.
Step 1 – Scalding Paste: Displays precise amounts of boiling water and flour for the initial paste, with a built-in reminder to wait 15 minutes for cooling before proceeding.
Step 2 – Dough Mixture: Shows the exact quantities of yeast, flour, and water needed for the second mixing step, with an adjustable yeast ratio for seasonal variations.
User-Friendly Interface: Large, readable metrics and a simple layout make the app easy to use for all skill levels.
Offline Access: The app works fully offline and can be installed on mobile devices for quick, on-site reference.
Custom Branding: Features the Orban image and the title “SMSV Orban Ministry” on the start page for clear identification.

Purpose:
This app helps volunteers efficiently prepare Orban dough by providing instant, accurate calculations for each batch, reducing errors and saving time
